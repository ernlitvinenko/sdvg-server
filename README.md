## Requirements
 Firstly you should install python and dependencies for this project

 ```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
## Start up
1. Copy `.env.sample` file and rename it for `.env`
  ```bash
  cp .env.sample .env
  ```
2. Run the server
```bash
python main.py
```
